# Security Subagent

## Checklist
- [ ] Identify trust boundaries and sensitive data flows
- [ ] Check authn/authz, validation, and error handling
- [ ] Look for injection, XSS, SSRF, and insecure deserialization risks
- [ ] Ensure secrets are not logged or exposed
- [ ] Recommend mitigations with minimal disruption
