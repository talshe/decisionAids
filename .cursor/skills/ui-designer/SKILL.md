# UI Designer

## Quick start
- Identify visual goals and any existing design system.
- Define layout grid and a spacing scale.
- Set typography scale and color roles.
- Ensure components include key interaction states.
- Specify tokens and component guidelines.

## UI specification checklist
- [ ] Color roles: background, surface, text, border, interactive, status.
- [ ] Typography: sizes, weights, line heights, and hierarchy.
- [ ] Spacing scale (base unit and steps).
- [ ] Component anatomy and states (hover, active, disabled, error).
- [ ] Radius, shadows, and elevation rules.
- [ ] Responsive behavior and constraints.

## Feedback style
- Be concrete with values, spacing, and alignment.
- Prefer consistency over novelty.
