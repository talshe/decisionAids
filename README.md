# decisionAids

A decision-aid platform skeleton built with React, Express, Firebase Authentication, and Firestore.

## What is included

- Public `Explore` screen and guest-friendly decision-aid player
- Authenticated `My Decision Aids` experience
- Admin workspace for creating, publishing, and assigning decision aids
- Backend APIs for discovery, favorites, assignments, and saved responses
- Firebase and Google Cloud deployment scaffolding
- Vitest-based frontend and backend test setup

## Project structure

- `frontend/`: React + Vite application
- `backend/`: Express API
- `docs/architecture.md`: domain and route overview
- `docs/deployment.md`: deployment and environment setup notes
- `infra/terraform/`: Cloud Run infrastructure-as-code
